## **Taskemer: Your Personal Task Management Assistant**

Taskemer is a sleek and intuitive task management application designed to help you stay organized and focused. Built with Python and PySide6, it offers a clean and modern user experience for tracking your tasks.

**Key Features:**

* **Modern User Interface:** Enjoy a clean and modern design with a card-based dashboard for easy task management.
* **Secure User Authentication:** Protect your data with a secure login and registration system.
* **Quick Task Creation:** Add new tasks effortlessly with a structured form that includes fields for name, description, and priority.
* **Clear Prioritization:** Assign priority levels to tasks with a clear visual indicator (low, medium, high) to focus on what's important.
* **Activity Logging:** Keep track of progress by logging work done on each task. View a history of logs with dates and descriptions.
* **Edit & Update Tasks:** Easily modify task details such as the name and description through a dedicated dialog.
* **Task Status Management:** Mark tasks as "Finished" or "Removed" directly from the dashboard with a single click.
* **Archived Tasks View:** Access a history of all your completed and removed tasks in a separate, organized view.
* **File Attachments:** Attach files, folders, or hyperlinks to your tasks and activity logs for quick reference.
* **Portable Database:** Your data is stored in a local SQLite database (`taskemer.db`), making your application portable and easy to back up.
* **Cross-Platform Potential:** While currently packaged for Windows, the underlying technology (Python/PySide6) allows for potential cross-platform compatibility.

**Getting Started:**

1.  **Installation:** Download the latest installer (`Taskemer_Setup_v1.0.exe`) and run it to install Taskemer on your Windows machine.
2.  **Registration:** Launch the application and create a new user account.
3.  **Add Tasks:** Use the "Add Task" view to create your first tasks.
4.  **Manage Your Tasks:** Track your progress, log activities, and mark tasks as complete on the dashboard.

**Built With:**

* **Python:** The core programming language.
* **PySide6:** The Qt for Python framework used for building the user interface.
* **SQLite:** The lightweight, disk-based database for storing your data.
* **Inno Setup:** Used to create the professional Windows installer.
